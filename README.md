# 台风风廓线模型参数查询系统

基于中国东南部对流层低层 (0–2000 m) 台风风速剖面模型研究

## 功能特点

- 台风风廓线参数查询
- 距离海岸线距离查询
- 地表覆盖类型查询
- 实时访问统计
- 响应式设计

## 部署说明

本项目使用 Vercel 进行部署，提供永久固定的网址。

## 开发

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建
npm run build

# 预览构建结果
npm run preview
```
